# 🥚 蛋仔乐园 - 手机网页游戏

一款类似《蛋仔派对》的可爱物理闯关游戏，专为手机浏览器优化。

## 🎮 游戏特点
- **可爱蛋仔角色**：多种颜色选择，丰富表情动画
- **物理闯关玩法**：跳跃、收集、躲避障碍
- **手机触控优化**：虚拟摇杆 + 大按钮，防误触设计
- **离线可玩**：HTML5技术，无需下载安装
- **适合小学生**：简单操作，明亮色彩

## 📱 如何在手机上玩

### 方法一：GitHub Pages（推荐）
1. 将本仓库上传到GitHub
2. 启用GitHub Pages功能
3. 用手机访问生成的网址

### 方法二：本地服务器
```bash
# 在电脑上启动服务器
python -m http.server 8000

# 手机访问：http://电脑IP:8000
```

### 方法三：直接打开
将 `index.html` 文件发送到手机，用浏览器打开

## 🛠️ 技术信息
- **游戏引擎**：纯HTML5 + CSS3 + JavaScript
- **物理效果**：CSS动画模拟
- **响应式设计**：支持各种手机屏幕
- **资源加载**：CDN引入，无需额外下载
